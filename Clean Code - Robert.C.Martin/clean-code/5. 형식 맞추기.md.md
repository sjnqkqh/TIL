### 코드 형식은 중요하다!

- 너무 중요해서 융통성 없이 맹목적으로 따르는 것은 지양해야 한다. 코드 형식은 의사소통의 일환임을 인지해야한다.

- 오늘 구현한 코드의 기능은 다음 버전에서 바뀔 가능성이 높다. 하지만 오늘 구현한 코드의 가독성은 앞으로 바뀔 코드의 품질에 지대한 영향을 끼친다.
